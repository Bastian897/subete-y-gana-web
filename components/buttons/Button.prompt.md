Primary CTA and action button for Súbete y Gana — gold gradient, secondary outlined, or ghost style.

```jsx
<Button variant="primary" size="lg">Participar ahora</Button>
<Button variant="secondary">Ver el premio</Button>
<Button variant="ghost" size="sm">Iniciar sesión</Button>
<Button variant="primary" disabled>Agotado</Button>
<Button variant="primary" fullWidth>Comprar participación</Button>
```

Variants: `primary` (gold fill + glow), `secondary` (gold outline), `ghost` (subtle outline).
Sizes: `sm`, `md`, `lg`.
Use `primary` for the main purchase CTA. Use `secondary` for supporting CTAs. Use `ghost` for navigation-level actions (login, back).
