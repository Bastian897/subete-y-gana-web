Live countdown timer displaying days / hours / minutes / seconds until the sorteo draw date.

```jsx
<Countdown targetDate="2025-09-15T20:00:00-04:00" />
```

Updates every second via setInterval. Clamps to 0 (never goes negative). Use in the hero section below the prize headline to drive urgency. Pair with a gold "Participar ahora" Button directly below.
